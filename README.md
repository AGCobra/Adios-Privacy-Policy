# Adios Privacy Policy

Public hosting for the [Adios](https://github.com/AGCobra/Adios) privacy policy, served via GitHub Pages:

**https://agcobra.github.io/Adios-Private-Policy/**

This is the URL for the App Store Connect "Privacy Policy URL" field.

The canonical text lives in the Adios repo (`PRIVACY_MODEL.md` and the in-app
`PrivacyPolicyView.swift`). All three copies must stay identical — when the
policy changes, update them in the same release and revise the effective date.
