# Adios Privacy Policy

Public hosting for the [Adios](https://github.com/AGCobra/Adios) privacy policy, served via GitHub Pages:

**https://agcobra.github.io/Adios-Privacy-Policy/**

This is the URL for the App Store Connect "Privacy Policy URL" field, and the
in-app "Privacy Policy" link (Settings ▸ About) points here too.

The canonical text lives in the Adios repo (`PRIVACY_MODEL.md`). The two
copies — that file and this site's `index.html` — must stay identical; when
the policy changes, update both in the same release and revise the effective
date. Do not break this URL: the shipped app links to it.
